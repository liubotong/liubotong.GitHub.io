本文将介绍如何利用github pages搭建个人博客

##前言

###你需要做的准备工作
1.注册github账号
2.安装git(在安装homebrew之后)

3.安装node.js
4.安装npm,node

##搭建github博客
1.创建仓库
仓库名称为username.github.io，即如果你的github用户名是xixi，则仓库名为xixi.github.io，配置好后，访问xixi.gihub,io就能看到你的网站啦～

##配置SSH key
为了更加安全，这里使用SSH key解决本地和服务器连接问题
1.首先查看本机是否已有ssh key

ssh key名字为rsa或dsa

2.如果没有，手动生成

3.生成之后，利用cat命令查看公钥，并打开你的github主页，进入个人设置 -> SSH and GPG keys -> New SSH key添加公钥
4.测试是否成功

出现提示*Are you sure you want to continue connecting (yes/no)?*输入yes后得到提示*You've successfully authenticated, but GitHub does not provide shell access.*说明配置成功
